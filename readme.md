## 📌 Project Overview

This portfolio showcases my academic journey, technical competencies, and professional goals at the intersection of quantitative finance and technology. The design strictly adheres to a minimalist, dark-themed aesthetic inspired by the **Kawsar Design Reference**, maintaining a highly professional and clean user interface. 

Crucially, this project was built **without the use of any external CSS frameworks** (like Bootstrap or Tailwind). All layouts, components, and responsiveness were engineered from scratch.

## 🚀 Features & Pages

The website consists of 7 fully interlinked HTML pages:
- **`index.html` (Home):** A high-impact hero section with a brief introduction and quick navigation calls-to-action.
- **`profile.html` (Profile):** A detailed biography detailing my background in FinTech, leadership roles (e.g., GenerationFAST Snow Leopards Foundation), and personal interests.
- **`gallery.html` (Gallery):** A CSS Grid-based visual collection of campus life, events, and personal milestones.
- **`goals.html` (Goals):** A split-layout page outlining my short-term academic targets and long-term career ambitions in quantitative finance and Explainable AI.
- **`projects.html` (Projects):** A showcase of technical projects (Blockchain, Econometrics, Excel Modeling) presented in a responsive card layout.
- **`skills.html` (Skills):** A visual breakdown of my technical proficiencies (Python, Data Mining, Web Engineering) using custom CSS progress bars.
- **`contact.html` (Contact):** A functional-looking, styled contact form and professional contact information.

## 🛠️ Technologies Used

- **HTML5:** Semantic markup structure.
- **CSS3:** - **Flexbox:** Used heavily for the navigation bar, profile alignment, and contact form layout.
  - **CSS Grid:** Utilized for creating uniform, responsive grids in the Gallery and Projects pages.
  - **Custom Styling:** Custom color palettes (`#0a0a0a` dark background, `#ffffff` text, `#a0a0a0` accents) and typography (Inter font).

## 📱 Responsive Design

The portfolio is fully responsive and guarantees a seamless user experience across all devices. The responsiveness is handled via custom Media Queries in a dedicated `responsive.css` file with the following breakpoints:
- **Desktop:** 1025px and above (Default Layout)
- **Tablet:** 641px – 1024px (Adjusts grids to 2 columns, stacks profile headers)
- **Mobile:** Up to 640px (Forces single-column layouts, stacks navigation links, adjusts typography for readability)
